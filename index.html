API de Nettoyage de Données CSV (Data Cleaning)

1. Déduplication (Statistiques JSON)

Supprime les lignes dupliquées et retourne le nombre de lignes avant/après.

2. Imputation des Manquants (Statistiques JSON)

Remplace les valeurs manquantes numériques par la médiane (retourne le décompte).

3. Suppression des Aberrants (Statistiques JSON)

Applique la règle `revenu < 20000` et `age > 0`.

4. NETTOYAGE COMPLET (CSV TÉLÉCHARGEMENT)

Effectue toutes les étapes (Déduplication -> Imputation -> Aberrants) et vous **téléchargez** le fichier corrigé.
